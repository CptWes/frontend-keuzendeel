# Frontend Verdieping Keuzendeel Project

## Description
This project is a Vue 3 web application I built with Vite.
It includes for now 3 pages and uses Vue Router for navigating them.
The project is deployed on GitHub Pages.

## Tech
- Vue

- Vite

- Vue Router 

- JSON

## Installation
```bash
npm install
```
## Development
```bash
npm run dev
```
## Build
```bash
npm run build
```
## Deployment
```bash
npx gh-pages -d dist -f
```
